
# OpenGL 2D Drawer

![App Screenshot](https://github.com/gracida001/opengl-2d-draw/blob/main/example.png)

## Deployment

To deploy this project run

```bash
  gcc main.c LibDraw/libdraw.c LibMenu/libmenu.c LibMath/libmath.c -l glut -l GL -o execute -Wall
```


```bash
   ./execute 
```
